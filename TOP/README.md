# 币安USDT合约涨跌幅排行榜

这是一个实时显示币安USDT交易对合约涨跌幅排名前十的桌面应用程序。

## 功能特点

- 实时获取币安USDT合约交易对的价格数据
- 计算并显示24小时涨跌幅
- 自动排序并展示涨幅和跌幅前十的交易对
- 美观专业的用户界面
- 数据自动刷新

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```

## 注意事项

- 需要稳定的网络连接以获取实时数据
- 数据每30秒自动刷新一次
