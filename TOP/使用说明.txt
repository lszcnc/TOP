币安USDT合约涨跌幅排行榜 - 使用说明
============================

【软件介绍】
这是一个实时显示币安USDT合约交易对涨跌幅排名的桌面应用程序。
程序会自动获取币安交易所的数据，并在同一个页面上展示涨幅和跌幅前10的交易对。

【功能特点】
1. 实时获取币安USDT永续合约交易对的价格数据
2. 智能过滤，只显示实际存在且正在交易的合约
3. 计算并显示24小时涨跌幅
4. 自动排序并展示涨幅和跌幅前十的交易对
5. 美观专业的用户界面
6. 数据每30秒自动刷新一次
7. 支持手动刷新功能，随时获取最新数据

【数据说明】
1. 本程序只显示币安合约市场中状态为"交易中"的USDT永续合约
2. 过滤掉了成交量为0的交易对
3. 界面顶部会显示当前获取到的有效交易对数量
4. 如果某些交易对未显示，可能是因为它们不是永续合约或已被币安下架

【安装与运行】
方法一：直接运行
1. 双击 run.bat 文件即可启动程序
2. 首次运行时会自动安装所需依赖，请耐心等待
3. 程序会询问是否安装matplotlib（用于生成自定义图标），可以选择Y或N

方法二：手动安装依赖
1. 确保已安装Python 3.7或更高版本
2. 打开命令提示符或PowerShell，进入程序所在目录
3. 执行命令：pip install python-binance==1.0.16 PyQt5==5.15.7 pandas==1.5.3 requests==2.28.2
4. 如需自定义图标，还需执行：pip install matplotlib==3.7.1
5. 安装完成后，执行：python main.py

【使用方法】
1. 程序启动后会自动获取并显示数据
2. 涨幅榜和跌幅榜会在同一个页面上显示
3. 点击底部的"刷新数据"按钮可以手动刷新数据
   - 刷新过程中按钮会暂时禁用，防止重复点击
   - 刷新完成后按钮会自动恢复可用状态
4. 数据每30秒会自动刷新一次

【注意事项】
1. 使用本程序需要稳定的网络连接
2. 如遇到数据获取失败，请检查网络连接或稍后再试
3. 本程序仅供参考，不构成任何投资建议
4. 交易有风险，投资需谨慎
5. 如果不安装matplotlib，程序仍能正常运行，只是没有自定义图标

【常见问题】
Q: 程序无法启动怎么办？
A: 请确保已安装Python 3.7或更高版本，并已正确安装所有依赖包。

Q: 数据显示"获取数据出错"怎么办？
A: 请检查网络连接是否正常，或者币安API是否可访问。

Q: 为什么某些交易对没有显示？
A: 本程序只显示币安合约市场中状态为"交易中"的USDT永续合约，并且过滤掉了成交量为0的交易对。如果某个交易对不符合这些条件，它将不会显示在列表中。

Q: 点击刷新按钮后没有反应怎么办？
A: 刷新过程中按钮会暂时禁用，请等待刷新完成。如果长时间没有反应，可能是网络问题，请检查网络连接。

Q: 出现"无法生成图标"的警告怎么办？
A: 这是因为缺少matplotlib库，程序仍能正常运行。如需自定义图标，请安装matplotlib：pip install matplotlib

Q: 如何退出程序？
A: 点击窗口右上角的关闭按钮，会弹出确认对话框，点击"是"即可退出。
